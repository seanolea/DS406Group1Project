# DS406 Group 1 — Airline Safety Project

## Overview
Statistical analysis of the FiveThirtyEight article:
**"Should Travelers Avoid Flying Airlines That Have Had Crashes in the Past?"** by Nate Silver (2014).

We carry out our own assessment of the article's central claim using airline safety data
from 1985–2014 across 56 major commercial airlines.

## Research Questions
1. Is an airline's past crash rate a statistically significant predictor of its future crash rate?
2. Is Silver's composite safety score a defensible and reproducible statistical construction?
3. Does a country's GDP per capita explain airline safety better than the airline's own historical record?

## Repository Structure

- `scripts/` — R scripts for each analysis
- `plots/` — figures and visualisations output by scripts
